# BTD6-Flooded-valley-AFK-
This is a simple program that automatically places monkeys and starts. This is fully afk.

The resolution is 1366x768 and you will kind of have to move your screen around to fix it.

when using tiny task, go into the settings and click the continuous playback button then change the amount of times to a random number in the billions. Once done adjust your screen around til it clicks on the right buttons and use ctrl+shift+alt+p to start.
